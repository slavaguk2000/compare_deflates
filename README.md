# D5LY - Browser
# The fastest deflate compressor for browser

## Usage 
### You need 3 files:
####	-d5ly.js
####	-d5ly.wasm
####	-d5ly_cover.js

```html
<script src="d5ly_cover.js"></script>
<script src="d5ly.js"></script>
```

```javascript
compressedArray = d5ly_compress(sourceArray)
decompressedArray = d5ly_decompress(compressedArray)
```
